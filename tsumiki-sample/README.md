# tsumiki-sample
- ref: https://github.com/classmethod/tsumiki
- tsumikiとは
  - AI駆動開発支援フレームワーク
  - AI구동개발지원프레임워크
- `commands`
    - `npx tsumiki install`시에 만들어지는 `.claude/commands`
- `commands-kr`
    - `commands`의 내용을 한국어로 번역
- `SUMMARY.md`
    - `commands-kr` 디렉토리의 파일들을 요약한 문서
- `TRANSLATION_REPORT.md`
    - 번역 상태 확인 리포트
    - 여러번 확인 안해주면 부족한 부분이 너무 많다
